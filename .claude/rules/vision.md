# 비전 및 AI 로드맵

## 최종 목표

> 채팅 기반 AI Agent가 가치투자를 제안하고 컨설팅하는 개인 투자 어드바이저

### 사용자 경험
```
사용자: "AAPL 지금 살만해?"
AI Agent: "AAPL 현재 P/E 28.5로 역사적 평균(25) 대비 고평가 상태입니다.
          하지만 ROE 147%, Net Margin 25%로 수익성은 매우 우수합니다.
          Graham Number 대비 20% 고평가이며, RSI 45로 중립 구간입니다.
          배당 수익률 0.5%로 배당 투자자에게는 매력이 낮습니다.

          💡 제안: 현재가보다 10% 하락 시 매수 고려"
```

## Phase 4: RAG 기반 AI 분석

### 목표
종목 데이터를 임베딩하여 자연어 질의에 응답

### 구현 계획

| 단계 | 작업 | 기술 |
|------|------|------|
| 4.1 | 종목 데이터 임베딩 | Claude/OpenAI Embeddings |
| 4.2 | 벡터 DB 구축 | Supabase pgvector (무료) |
| 4.3 | 검색 API | FastAPI + 유사도 검색 |
| 4.4 | LLM 연동 | Claude API |

### 질의 예시
- "저평가된 배당주 추천해줘"
- "삼성전자 지금 살만해?"
- "ROE 20% 이상, P/E 15 이하인 종목"

### 임베딩 대상 데이터
- 종목 기본 정보 (이름, 섹터, 시장)
- 재무 지표 (P/E, P/B, ROE 등)
- 기술적 지표 (RSI, MACD, 볼린저)
- 최근 뉴스/공시 (향후)

## Phase 5: AI 투자 어드바이저

### 목표
채팅으로 투자 상담을 받는 개인 어드바이저

### 인터페이스 옵션

| 채널 | 장점 | 단점 |
|------|------|------|
| Discord Bot 확장 | 기존 인프라 활용 | 사용자 제한 |
| 웹 채팅 UI | 접근성 높음 | 개발 비용 |
| Telegram Bot | 모바일 친화적 | 추가 개발 |

### 기능
- 종목 분석 요청 → RAG 검색 → LLM 분석 → 응답
- 포트폴리오 리밸런싱 제안
- 시장 동향 요약
- 맞춤형 종목 추천

### 에이전트 아키텍처
```
[사용자 질의]
    ↓
[의도 분류] → 종목 분석 / 스크리닝 / 일반 질문
    ↓
[RAG 검색] → Supabase pgvector
    ↓
[컨텍스트 구성] → 관련 종목 + 지표 + 뉴스
    ↓
[LLM 분석] → Claude API
    ↓
[응답 생성]
```

## 고려사항

### 비용
- Claude API: 토큰당 과금
- 임베딩: 초기 1회 + 주간 업데이트
- Supabase pgvector: 무료 티어 내 가능

### 법적 고지
- 투자 조언이 아닌 정보 제공 목적
- 면책 조항 필수 표시

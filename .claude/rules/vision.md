# 비전 및 AI 로드맵

## 최종 목표

> 채팅 기반 AI Agent가 가치투자를 제안하고 컨설팅하는 개인 투자 어드바이저

### 사용자 경험
```
사용자: "AAPL 지금 살만해?"
AI Agent: "AAPL 현재 P/E 28.5로 역사적 평균(25) 대비 고평가 상태입니다.
          하지만 ROE 147%, Net Margin 25%로 수익성은 매우 우수합니다.
          Graham Number 대비 20% 고평가이며, RSI 45로 중립 구간입니다.
          배당 수익률 0.5%로 배당 투자자에게는 매력이 낮습니다.

          💡 제안: 현재가보다 10% 하락 시 매수 고려"
```

## Phase 4: AI 종목 분석

### 목표
Claude Code CLI + 웹검색으로 종목별 AI 분석 생성, OpenSearch에 저장

### 아키텍처

```
[데이터 수집 완료]
    ↓
[Claude Code CLI] ← Max 구독 활용 (API 비용 $0)
    │ - 웹검색 (뉴스, 이슈)
    │ - 지표 분석
    │ - 인사이트 생성
    ↓
[OpenSearch] ← 직접 호스팅
    │ - 티커별 분석 문서 저장
    │ - 변경분만 업데이트
    ↓
[Stock-Screener API]
    │ - OpenSearch 조회만
    ↓
[프론트엔드]
```

### 구현 계획

| 단계 | 작업 | 기술 |
|------|------|------|
| 4.1 | OpenSearch 인프라 | AWS OpenSearch / 직접 호스팅 |
| 4.2 | 분석 스크립트 | Claude Code CLI + 웹검색 |
| 4.3 | 분석 결과 저장 | OpenSearch 인덱스 |
| 4.4 | 조회 API | FastAPI + OpenSearch 클라이언트 |

### 분석 내용
- 종목 기본 정보 + 재무 지표 요약
- 웹검색 기반 최근 뉴스/이슈
- 투자 인사이트 (매수/관망/회피 근거)
- 리스크 요인

### 질의 예시
- "AAPL 지금 살만해?" → 저장된 분석 조회
- "삼성전자 최근 이슈" → 웹검색 기반 뉴스 요약

## Phase 5: AI 투자 어드바이저

### 목표
채팅으로 투자 상담을 받는 개인 어드바이저

### 인터페이스
웹 채팅 UI (접근성 + 기존 프론트엔드 통합)

### 기능

| 기능 | 설명 |
|------|------|
| 오늘의 추천 | 저평가 종목 자동 추천 |
| 개인화 추천 | 워치리스트 기반 유사 종목 |
| 채팅 UI | 자연어 질의 응답 |
| 시장 동향 | 일일 시장 요약 |

### 에이전트 아키텍처
```
[사용자 질의]
    ↓
[의도 분류] → 종목 분석 / 스크리닝 / 추천 / 일반
    ↓
[OpenSearch 검색] → 저장된 분석 조회
    ↓
[컨텍스트 구성] → 관련 종목 + 지표 + 뉴스
    ↓
[LLM 응답 생성] → Claude Code CLI
    ↓
[웹 채팅 UI]
```

## 고려사항

### 비용 전략
- **Phase 4**: Claude Code CLI (Max 구독) → **$0**
- **OpenSearch**: 직접 호스팅 또는 AWS Serverless
- **Phase 5**: 실시간 채팅은 Claude API 필요할 수 있음

### 분석 대상 범위 (의사결정 필요)
- 전체 ~9,000개: 시간/리소스 많이 소요
- 워치리스트 종목만: 개인화되지만 제한적
- 시가총액 Top N: 균형잡힌 접근

### 법적 고지
- 투자 조언이 아닌 정보 제공 목적
- 면책 조항 필수 표시

```
본 정보는 투자 참고용이며 투자 조언이 아닙니다.
AI 분석은 참고 자료로만 활용하시고, 투자 결정은 본인의 판단과 책임 하에 이루어져야 합니다.
```

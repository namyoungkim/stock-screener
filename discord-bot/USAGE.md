# Stock Screener Bot 사용법

주식 정보 조회, 워치리스트 관리, 알림 설정을 할 수 있는 봇입니다.

---

## 종목 조회

```
/stock AAPL          # 애플 정보 조회
/stock 005930        # 삼성전자 정보 조회
/search 테슬라        # 종목 검색
```

---

## 스크리닝

```
/presets             # 프리셋 목록 보기
/screen graham       # 그레이엄 전략으로 스크리닝
/screen buffett      # 버핏 전략으로 스크리닝
/screen dividend     # 고배당주 스크리닝
/screen deep_value   # 저평가주 스크리닝
```

---

## 워치리스트

나만의 관심 종목을 저장하고 관리할 수 있습니다.

```
/watch AAPL          # 워치리스트에 추가
/watchlist           # 내 워치리스트 보기
/unwatch AAPL        # 워치리스트에서 삭제
```

---

## 알림 설정

특정 조건이 충족되면 알림을 받을 수 있습니다.

### 알림 생성

```
/alert AAPL pe_ratio <= 15       # P/E가 15 이하일 때
/alert AAPL rsi <= 30            # RSI가 30 이하일 때 (과매도)
/alert 005930 latest_price <= 70000  # 삼성전자 7만원 이하일 때
```

### 알림 관리

```
/alerts              # 내 알림 목록
/delalert {id}       # 알림 삭제
/togglealert {id}    # 알림 켜기/끄기
```

### 사용 가능한 지표

| 지표 | 설명 |
|------|------|
| `pe_ratio` | P/E 비율 |
| `pb_ratio` | P/B 비율 |
| `roe` | 자기자본이익률 (%) |
| `rsi` | RSI (0-100) |
| `dividend_yield` | 배당수익률 (%) |
| `latest_price` | 현재가 |
| `graham_number` | 그레이엄 넘버 |

### 비교 연산자

| 연산자 | 의미 |
|--------|------|
| `<` | 미만 |
| `<=` | 이하 |
| `=` | 같음 |
| `>=` | 이상 |
| `>` | 초과 |

---

## 참고

- 각 사용자의 워치리스트와 알림은 **개인별로 저장**됩니다
- 다른 사람의 워치리스트/알림은 볼 수 없습니다
- 한국 주식은 숫자 티커 사용 (예: `005930`)
- 미국 주식은 영문 티커 사용 (예: `AAPL`)
